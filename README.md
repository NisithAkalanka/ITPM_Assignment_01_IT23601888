# ITPM Assignment 01 - UI Automation with Playwright

This repository contains automated UI test scripts designed to evaluate the functionality of the **PixelsSuite** web application. This project was completed as part of the IT3040 - Information Technology Project Management unit.

## Features Automated
- **Chat Sinhala Transliteration Accuracy Testing:** Testing various categories of Singlish inputs (informal Sinhala typing) and validating the transliteration correctness on the [PixelsSuite Chat-Translator](https://www.pixelssuite.com/chat-translator).

## Technologies Used
- **Programming Language:** Python 3.12+
- **Automation Framework:** Playwright
- **Reports/Data:** Microsoft Excel (openpyxl)

## Prerequisites
Ensure you have Python installed on your system.
```bash
python --version
```

## Setup and Installation

1. **Install required Python packages:**
   ```bash
   pip install playwright openpyxl
   ```

2. **Install Playwright Browsers (Chromium):**
   ```bash
   playwright install chromium
   ```

## How to Run the Tests

To run the transliteration accuracy tests and update the Excel results:

```bash
python test_automation.py --excel "Assignment 1 - Test cases.xlsx" --url "https://www.pixelssuite.com/chat-translator" --wait-ms 5000 --type-delay-ms 80 --slow-mo-ms 200 --save-every 1
```

## Test Results
After execution, the results are automatically updated in the `Assignment 1 - Test cases.xlsx` file:
- **Actual Output:** The Sinhala translation generated by the system.
- **Status:** Marks "PASS" if the output matches the expected value, otherwise "FAIL".
